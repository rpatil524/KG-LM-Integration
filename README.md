# KG-LM-Integration
This is the implementation of KIG-Bert proposedo by Aisha and Xiangru in the CS 848 course project.

## Usage
All the experimental results can be reproduced by the jupyter notebook KIG-Bert.ipynb. Detailed documentation and instruction is in the notebook.
 
## Requiremnts  
1. GPU is needed for training and evaluation.
2. Git Large File Storage package is needed. Please find the intrustion on the installing it on https://docs.github.com/en/repositories/working-with-files/managing-large-files/installing-git-large-file-storage
